# 📝 Markdownチートシート

### 1. 見出し
# 見出し1
## 見出し2
### 見出し3

### 2. テキスト装飾
- **太字**: `**テキスト**`
- *斜体*: `*テキスト*`
- ~~打ち消し線~~: `~~テキスト~~`

### 3. 箇条書き
- 箇条書き1
    - ネスト（行頭で半角スペース2つ）
1. 番号付き1
2. 番号付き2

### 4. チェックボックス（タスク管理）
- [ ] 未完了のタスク
- [x] 完了したタスク

### 5. 引用と区切り線
> 引用文です。
> 改行しても引用が続きます。

---
（↑ `---` で水平線）

### 6. コード表示
インラインで `code` を書く場合は `` `code` ``。

複数行のコードブロック：
```javascript
function test() {
  console.log("Hello World");
}
